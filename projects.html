Projects

Below are side projects which I have undertaken or may still be in development.

Project Suggestion Site

This website allows users to register, login and post project suggestions which must be approved by an admin in order to be displayed. It uses Flask as its web-framework and a PostgreSQL database to store users details such as username, password (hashed), and role as well as post details such as title, description and whether it is approved. View Code.

Book Inventory Manager

A simple Java class based program to model a book invetory manager, the book class holds attributes which define a book e.g. title, author, ISBN etc. it also implements the comparable interface to easily sort the books by release year. The inventory class contains a list of books with comparators to help sort the list of books, it can also perform basic calculations such as total inventory value. View Code.
