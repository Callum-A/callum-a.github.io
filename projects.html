Book Inventory Manager

A simple Java class based program to simulate a book inventory manager. Features two main classes representing a book and an inventory of books. The book class contains attributes used to define a book e.g. Title, Author, ISBN etc. also implements the comparable interface to sort the books by release year. The inventory class contains a list of books.
